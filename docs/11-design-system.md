# 11. Design System

## Direcao visual
- Tipografia: `Sora` para heading e `Manrope` para corpo.
- Paleta base: cream, slate e amber.
- Cards com vidro claro e paineis dark para performance.
- Booking com foco mobile-first.

## Tokens
- definidos em `packages/ui/src/styles.css`
- superficie clara: `bf-surface`
- superficie dark: `bf-surface-dark`
- mesh panel: `bf-mesh-panel`

## Componentes compartilhados
- `Button`
- `Card`
- `DarkCard`
- `Badge`
- `Input`
- `Textarea`

## Regra
- marketing e app compartilham a mesma linguagem.
- sem hotlinks para assets visuais.
- usar assets locais em `apps/web/public`.
